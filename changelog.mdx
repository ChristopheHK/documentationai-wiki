## v1.3.0 - October 15, 2024

<Update label="2024-10-15" description="v1.3.0" tags={["feature", "improvement"]}>

## New Features

- Added advanced search filters in the mediator directory, including expertise areas and location preferences.
- Introduced real-time case status notifications via email and in-app alerts.

## Improvements

- Enhanced mobile responsiveness for the dashboard, ensuring seamless access on all devices.
- Optimized request submission flow, reducing steps by 20%.

## Bug Fixes

- Resolved intermittent login issues for mediators.
- Fixed display errors in case history timelines.

</Update>

## v1.2.0 - September 20, 2024

<Update label="2024-09-20" description="v1.2.0" tags={["bugfix", "security"]}>

## Improvements

- Upgraded security protocols for user data handling and session management.

## Bug Fixes

- Corrected file upload errors during mediation request submissions.
- Fixed search results pagination in the mediator annuaire.
- Addressed accessibility issues for screen reader users.

</Update>

## v1.1.0 - August 10, 2024

<Update label="2024-08-10" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- Launched integrated payment system for mediation fees directly in the platform.
- Added multi-language support for French and English interfaces.

## Breaking Changes

- Updated API endpoints for request submissions; see migration guide below.
